# Smart Pizzaria (PHP + MySQL)

## 1) Base de données
- Importer le fichier: `db/smart_pizzaria.sql`
- DB name: `smart_pizzaria`

## 2) Config
Modifier `config/config.php` si nécessaire (user/pass MySQL).

## 3) Lancer
- Mettre le dossier dans `htdocs` (XAMPP) ou lancer le serveur PHP:
  - `php -S localhost:8000`
- Ouvrir `http://localhost:8000/index.php`
